# TruthLens Privacy Policy

Last updated: May 2026

## What we collect
TruthLens sends the text of individual sentences to our verification 
backend for fact-checking. No other data is collected.

## What we do NOT collect
- No personal information
- No browsing history  
- No login credentials
- No IP addresses stored permanently
- No cookies

## How data is used
Sentence text is sent to our backend server which queries Wikipedia, 
PubMed, and Wikidata. Results are cached in our database to improve 
speed for future users. Cached data contains only the sentence and 
its verification result — no user identifiers.

## Third party services
- Groq API (claim classification)
- Wikipedia REST API (fact verification)
- PubMed NCBI API (medical fact verification)
- Wikidata API (structured fact verification)
